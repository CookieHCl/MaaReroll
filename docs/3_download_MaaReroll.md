# 3. MaaReroll 다운로드 및 실행

## 3.1. MaaReroll 다운로드

[Release](https://github.com/CookieHCl/MaaReroll/releases) 페이지에서 자신의 OS에 맞는 압축파일을 받아주세요.  
대부분 유저의 경우 **MaaReroll-win-x86_64**로 시작하는 파일을 고르면 됩니다.

그 후 압축을 해제한 다음 `MFAAvalonia.exe`를 실행하세요.

![잘못된 에러팝업](/docs/image/3/not_error.png)

오른쪽 아래에 이 팝업이 뜨는건 에러가 아니니 팝업을 클릭해 무시하시면 됩니다.

## 3.2. 명일방주 연결

![명일방주 연결](/docs/image/3/connect_arknights.png)

1. 일단 프로그램을 키자마자 무조건 오른쪽 위의 새로고침 버튼을 눌러주세요.
1. 연결하고 싶은 앱플레이어 인스턴스를 골라주세요.
1. 오른쪽 위의 연결 버튼을 눌러주세요.

![명일방주 연결 완료](/docs/image/3/arknights_connected.png)

연결이 잘 됐다면 오른쪽 위의 연결 아이콘이 초록색으로 뜨며, Live view로 실시간 화면을 볼 수 있습니다.

### 3.3.1. 배럭용 명일방주 연결

![Live view 종료](/docs/image/3/turn_off_live_view.png)

우선 왼쪽 아래 살정 화면에 들어간 다음 *Running - Live view*에서 사양을 잡아먹는 Live view를 끄세요.

![MaaReroll 폴더 복사](/docs/image/3/copy_folder.png)

그 후 반드시 **열려있는 MaaReroll을 종료한 다음** MaaReroll 폴더를 복사해주세요.

> [!TIP]
> ![MaaReroll 폴더 이름 변경](/docs/image/3/rename_folder.png)
>
> 폴더를 전부 드래그한 다음 `F2`를 눌러 이름을 변경하면 한번에 폴더 이름을 변경할 수 있습니다.

![명일방주 배럭 연결](/docs/image/3/connect_multiple_arknights.png)

이후 각 폴더마다 `MFAAvalonia.exe`를 실행하고 서로 다른 instance들을 연결해주면 됩니다.  
반드시 `MFAAvalonia.exe`를 한 폴더마다 하나씩 실행해야 합니다.

## 3.3. 매크로 시작

![매크로 시작](/docs/image/3/start_maareroll.png)

1. 왼쪽에서 리세마라 Task가 선택되어 있는지 확인해주세요.
1. 가운데의 Task Settings를 설정해주세요. ([4.](/docs/4_setup_MaaReroll.md)를 참고해주세요)
1. 오른쪽 위의 Start Task를 누르세요.

정상적으로 동작할 경우 계속해서 계정 생성-탈퇴를 반복하며 오퍼레이터가 뜰 때까지 매크로가 돌아갑니다.
