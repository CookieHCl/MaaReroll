# 4. MaaReroll 설정

[4.1. 리세 방식](#41-리세-방식)과 [4.2. 현재 화면](#42-현재-화면)은 반드시 읽어보세요.

## 4.1. 리세 방식

- 0-1 퇴각런: 가장 빨리 가챠를 돌리는 방법으로, 0-1에서 퇴각합니다.
- 0-1 클리어런: 0-1을 클리어한 다음 TR-1에서 퇴각해 1뽑을 더 얻습니다.
- 0-11 클리어런: 모든 이벤트 보상을 받는 방법으로, 0-11까지 클리어합니다.

보통 0-1 퇴각런이 제일 효율이 좋고, 0-11 클리어런은 오래 걸리지만 가챠를 가장 많이 돌릴 수 있습니다.  
뿌리는게 없거나, 1뽑만 더 받으면 10뽑을 돌릴 수 있는 경우 0-1 클리어런이 0-1 퇴각런보다 효율이 더 좋을 수도 있습니다.  
(단뽑은 캐릭터 대사를 기다려야 해서 10뽑보다 훨씬 오래 걸림)

## 4.2. 현재 화면

현재 매크로 화면을 선택합니다. 특정 지점부터 매크로를 실행할 수 있습니다. 정확한 시작 지점은 아래 표를 참고하시기 바랍니다.

보통 매크로가 잘 돌아가면 맨 처음에만 로그인으로 돌리고 (종료 조건을 만족해서) 매크로가 끝난 이후에는 오퍼레이터로 돌립니다.  
오퍼레이터에서 시작 -> 매크로 종료 -> 뽑은거 확인한 다음에 마음에 안 들면 오퍼레이터에서 다시 시작 -> 반복

- 로그인: 맨 처음 로그인 화면입니다.
- 프롤로그 (1번째 전투): 이름 선택/스토리 시작/전투 시작할 때입니다.
- 프롤로그 (2번째 전투): 2번째 스토리 시작/전투 시작할 때입니다.
- 막간 (메인화면 설정): 0-1 진입 전 메인화면에서 PRTS가 여러 가지 설명해줄 때입니다.
- 홈 (가챠 진행): 즉시 헤드헌팅으로 들어가서 가챠를 돌립니다. **메일 등 가챠 재화는 미리 받으셔야 합니다.**
- 오퍼레이터 (계정삭제 진행): 즉시 계정을 탈퇴한 다음 새 계정을 만들어서 매크로를 돌립니다.

> [!TIP]
> 매크로 오작동으로 인해 중간에 멈췄을 경우 위 화면들 중 하나로 수동으로 이동한 뒤 현재 화면을 설정한 후 매크로를 다시 실행하면 됩니다.
>
> 0-1에 도달하지 못 했을 경우 **로그인/프롤로그/막간** 중 하나를 골라주세요.
> 0-1에 도달했을 경우 어디서 멈췄든간에 일단 퇴각하고 홈에서 필요한 가챠 재화를 전부 받아준 다음 **홈**을 골라주세요.

> [!CAUTION]
> 이전 계정을 탈퇴하지 않은 채로 로그인 화면에서 시작하면 이전에 사용했던 게스트 계정이 자동으로 복구돼서 매크로가 꼬이는 경우가 생깁니다.
>
> 이 경우 일단 로그인을 한 다음 알맞은 현재 화면을 고르고 다시 실행해주세요.

## 4.3. 종료 조건

종료 조건은 크게 개수 조건과 오퍼레이터 조건으로 나뉩니다.

- 개수 조건은 6성이 몇 개 이상 떴을 때 종료하는 조건입니다.
- 오퍼레이터 조건은 특정 6성이 떴을 때 종료하는 조건입니다.

종료 조건을 **AND**로 설정하면 두 조건을 모두 만족했을 때, **OR**로 설정하면 두 조건 중 하나라도 만족했을 때 종료합니다.

## 4.4. 픽업 배너

리세를 돌릴 픽업 배너를 정합니다.  
보통 두 번째 배너가 가장 뽑고 싶은 신캐 배너이므로 이걸 고르시면 됩니다.

c.f. 일반적인 배너 순서는 신규뽑 - 신캐 - 복각 - 커널

## 4.5. 순오리지늄 사용

극한의 이득을 보고싶다면 스킨에도 쓸 수 있는 순오리지늄을 뽑기에 사용하지 않을 수도 있습니다.  
근데 어차피 맵 깨면 순오리지늄 퍼주고 이거 꺼서 리세 더 오래 돌리는게 손해라 그냥 순오리지늄 쓰는게 낫습니다.

## 4.6. 직접 뽑기

뽑기를 직접 해보고 싶은 경우 키시면 됩니다.

이 옵션을 킬 경우 **종료 조건을 무시**하고 무조건 뽑기 화면에서 멈춥니다.
